ABOUT botREC

botREC is a waste classification device powered by artificial intelligence. It is designed to make recycling easy and fun, and to promote green practices in residential areas.

Key features of botREC

Portable

Powered by a single-board computer, botREC is small and easily installable.

Easy to use

With the push of a button, botREC uses LEDs to notify indivudials of the correct trash category

Integrable

Requiring essentially no changes to the current waste collection system, botREC is very easy to adopt.

Sustainable

botREC can be solar powered when used outdoors, making it environmentally friendly
